Scrabble

The Scrabble game was my first attempt in creating a game from scratch with Python. The single player game randomly selects a letter that your must start with. The word that you make is then scored according to the scrabble scoring system. If your word didn't start with the letter selected then it was not scored and a turn was added to turn counter. You have to try to get to 150 points as fast as possible using as little amount of turns as possible. Each word entered is also checked against the Scrabble Dictionary to validate the word used. At the end the score was added to scoreboard.

In the photo of the Scrabble Game you can see what happens when you enter a misspelled or duplicate word, or even entering a word that doesn't start with the correct letter.
